# Litoral EasyBook - Platforma Web de Rezervari Online

Aplicatie Single Page Application (SPA) responsiva destinata rezervarilor de cazari pe litoralul romanesc, dezvoltata folosind HTML5, CSS3, Bootstrap 5 si JavaScript nativ (ES6+).

## Caracteristici cheie:
- Managementul a 4 roluri: Vizitator, Turist, Proprietar, Administrator.
- Persistenta totala a datelor client-side folosind Web Storage API (`localStorage`).
- Algoritm avansat pentru prevenirea rezervarilor suprapuse (Anti-Overlap).
- Suport integrat pentru butoanele Back/Forward din browser prin HTML5 History API.
- Securizare la inregistrare pentru functiile administrative.
